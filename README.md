# Alura-Store

Proyecto: Alura Store

Estado del proyecto: culminado

Descripción: Este proyecto surge como respuesta al **Challenge de Data Science de Alura Latam**. El objetivo principal es asesorar al dueño de una cadena de cuatro tiendas ("El Sr. Juan") para decidir qué unidad de negocio debe ser vendida. 

A través de un análisis profundo de datos de ventas, logística y satisfacción, se identificaron las fortalezas y debilidades de cada tienda para maximizar la rentabilidad de una futura inversión.

Tecnologías usadas:

*   **Lenguaje:** Python 3.12
*   **Manipulación de Datos:** Pandas
*   **Visualización Estática:** Matplotlib & Seaborn
*   **Entorno:** Google Colab 

Análisis:

El análisis se divide en 5 módulos clave:
1.  **Consolidación de Datos:** Limpieza y unificación de 4 bases de datos distintas con estandarización de columnas (tildes, espacios y tipos de datos).
2.  **Análisis Financiero:** Cálculo de ingresos totales por tienda para identificar el motor económico del negocio.
3.  **Popularidad de Productos:** Identificación de las categorías más vendidas y productos estrella vs. productos con baja rotación.
4.  **Satisfacción del Cliente:** Análisis de las calificaciones promedio (Rating) para medir la salud de la marca en cada ubicación.
5.  **Análisis Logístico:** Evaluación de los costos de envío y su impacto directo en la satisfacción del comprador.


Visualizaciones Destacadas

El proyecto incluye:
*   Gráficos de barras para comparación de ingresos.
*   Boxplots para entender la distribución de precios y calificaciones.
*   Scatter plots para analizar la relación entre costos de envío y satisfacción.
